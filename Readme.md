## Summary

These pages contain some materials for a *data analysis for neuroimaging* module [PSYG4043] at the University of Nottingham, a core module on the *MSc Cognitive Neuroscience*.

In the class, we acquire various MRI data sets - (emphasis on fMRI, anatomical, and/or DTI) in small groups. We then use a combination of tools to analyse the data.

The materials for the module are hosted on the Moodle pages for the module, but for the scanning / data acquisition days and for code examples, we use this github repository:

| Week | Unit                                                           | Topic                                                     |
| :--- | :------------------------------------------------------------- | :-------------------------------------------------------- |
| 2    | [Getting data](gettingData.md)                      | Data acquisition (sessions on 3T)   (DS)          |
| 4 (reading week!)   | [UNIX + git](unix-2-and-versions.md)                           | **self-paced, no class!** Version control, working in a UNIX environment (DS)              |
| 6    | [Making figures in Matlab](matlab-images.md)                  | Using Matlab to make publication-ready figures   (DS)     |
